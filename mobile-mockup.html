🎨 Agora Mobile UI Mockup

  • Smart Title Bar: Tap "Following" (or "Agora", "Agora Venezuela", etc.) to switch feeds - it's the main title now
  • Inline Hashtags: Followed hashtags scroll horizontally to the right of the title (single line, saves vertical space)
  • Optimized NoteCard: Avatar inline with name (saves ~25% horizontal space)
  • Touch-friendly: 44px minimum touch targets on all interactive elements
  • Bottom Navigation: 5 key sections in thumb-reach zone
  • Compose FAB: Floating action button for quick posting
  • Clean hierarchy: Name → Handle → Time flow naturally left-to-right
  • Full-width content: Text uses entire available width now

Try it: Tap "Following" to open feed selector. Tap hashtags to filter. Notice everything fits on one line!
